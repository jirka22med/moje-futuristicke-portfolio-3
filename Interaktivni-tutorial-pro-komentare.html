Průvodce komentáři v kódu

Naučte se psát poznámky v HTML, CSS a JavaScriptu jako profík.

HTML CSS JavaScript

HTML Komentáře

V této sekci se naučíte, jak vkládat komentáře přímo do vaší HTML struktury. Komentáře jsou skvělým způsobem, jak si zanechat poznámky o různých částech stránky, vysvětlit složitější kód nebo dočasně "vypnout" část kódu, aniž byste ji museli smazat. Prohlížeč tyto poznámky kompletně ignoruje a uživatel je na stránce nikdy neuvidí. 

<!-- Toto je komentář. Slouží pro poznámky přímo v kódu. -->

<div class="container">
    <h1>Nadpis stránky</h1>
    
    <!-- Zde začíná navigační menu -->
    <nav>
        <!-- Odkazy budou přidány později -->
    </nav>
</div>

Důležité: V HTML nesmíte vnořovat komentáře jeden do druhého. Každý komentář musí být samostatně otevřen (`<!--`) a uzavřen (`-->`).

CSS Komentáře

Zde se dozvíte, jak používat komentáře v CSS souborech. Komentáře ve stylech jsou nepostradatelné pro organizaci vašeho kódu. Můžete jimi oddělovat logické bloky stylů (např. pro hlavičku, tlačítka, patičku), vysvětlovat, proč jste použili konkrétní hodnotu, nebo si zanechat poznámku pro budoucí úpravy. Stejně jako v HTML, prohlížeč tyto komentáře ignoruje. 

/* Hlavní paleta barev */
:root {
    --primary-color: #3498db;
    --background-color: #f1f5f9;
}

body {
    background-color: var(--background-color);
    /* Nastavení základního písma pro celou stránku */
    font-family: 'Inter', sans-serif;
}

/* Styly pro všechna tlačítka */
.button {
    background-color: var(--primary-color);
    color: white; /* Barva textu */
}

JavaScript Komentáře

V této poslední části se seznámíte se dvěma typy komentářů v JavaScriptu. Jsou klíčové pro popis logiky vašich skriptů – co dělají jednotlivé funkce, jaký je účel proměnných nebo jak fungují složité algoritmy. JavaScript nabízí flexibilitu pro krátké i delší poznámky.

Jednořádkový komentář

Začíná se dvěma lomítky (`//`) a platí od nich až do konce řádku. Ideální pro krátké, rychlé poznámky.

// Deklarace proměnné pro uložení jména uživatele
let userName = "Jiřík";

// Vypíše pozdrav do konzole prohlížeče
console.log("Ahoj, " + userName);

Víceřádkový komentář

Začíná `/*` a končí `*/`. Všechno mezi tím je komentář, i když se rozprostírá přes více řádků. Hodí se pro detailní popis funkcí nebo složitějších částí kódu.

/*
 * Tato funkce sečte dvě čísla a vrátí jejich součet.
 * @param {number} a - První číslo
 * @param {number} b - Druhé číslo
 * @returns {number} Součet čísel a a b.
*/
function secti(a, b) {
    return a + b;
}
